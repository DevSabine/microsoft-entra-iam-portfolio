# Department-Based Access Control Lab (Microsoft Entra ID)

## Objective
Simulate access assignment by department using group-based access control.

## Implementation Details
- Created users for HR, IT, Finance
- Created groups for each department
- Assigned users to groups

## Skills Demonstrated
- Identity segmentation
- Group-based access control
- Access governance fundamentals

## Why It Matters
Organizations assign access based on department roles rather than individual users. 
This ensures consistency, scalability, and reduces security risk.

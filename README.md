# microsoft-entra-iam-portfolio
Microsoft Entra ID IAM portfolio documenting identity lifecycle management, RBAC, MFA, Conditional Access and governance labs.

## IAM Lab Environment Overview

This repository simulates a real-world enterprise identity and access management (IAM) environment using Microsoft Entra ID.

The environment includes:

- Department-based user structure (HR, IT, Finance)
- Role-based access control (RBAC)
- Multi-Factor Authentication (MFA) enforcement
- Conditional Access policy awareness
- Identity Governance (Access Reviews)
- Privileged Identity Management (PIM)
- Application access management

Each lab demonstrates how identity security is implemented, enforced, and monitored in a modern cloud environment.

## Labs Included

- 01 – Identity Lifecycle Management  
- 02 – Department-Based Access Control  
- 03 – Role-Based Access Control  
- 04 – Conditional Access  
- 05 – Multi-Factor Authentication (MFA)  
- 06 – Access Reviews (Identity Governance)  
- 07 – Privileged Identity Management (PIM)  
- 08 – Application Access Management
